ROWNUM
=========

IT WILL GENERATE THE SEQUENTIAL RECORD NUMBERS.
ROWNUM WILL BE GENERATED AT THE TIME OF QUERYING.
ROWNUM WILL BE GENERATED TILL CONDITION IS TRUE

SELECT ROWNUM,EMPNO,ENAME FROM EMP
WHERE DEPTNO=10;

ROWNUM	EMPNO	ENAME

1	7839	KING
2	7782	CLARK
3	7934	MILLER

ROWID
======
ITS A MEMORY LOCATION ADDRESS OF ROW
WHEN EVER WE  ARE INSERTING THE ROW INTO TABLE THEN THAT ROW WILL OCCUPY THE PHYSICAL MEMORY.
THIS MEMORY WILL BE REPRESENTED BY ROWID.
ROWID IS ALPHANUMERIC AND IT WILL CONTAIN 18 CHARACTERS

SELECT E.EMPNO,SUBSTR(E.ROWID,1) FROM EMP E
WHERE DEPTNO=10;

EMPNO	SUBSTR(E.ROWID,1)

7839	AAAnM4ABnAAAXn8AAA
7782	AAAnM4ABnAAAXn8AAC
7934	AAAnM4ABnAAAXn8AAN

SELECT EMPNO FROM EMP WHERE ROWID='AAAnM4ABnAAAXn8AAA';

7839
